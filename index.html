Projects

  • 🎨 Etch-A-Sketch 🖌️

    The Etch-A-Sketch Application turns your web browser into a limitless canvas for creativity. It starts with a key step - the “Resize” button, which lets you re-size your drawing area. This unlocks the app's full potential, revealing five tools to inspire your imagination. You can pick from three vibrant colors (black, grey, and rainbow), use the eraser to fix mistakes, and hit clear to start over. Whether you're a beginner or an experienced artist, this app offers a simple yet powerful set of tools to create digital art.

    Tools used: HTML, CSS, and JavaScript

  • 📚 Library 📖

    The Library application is a web application that helps you organize and manage your book collection. It's designed to make cataloging your books easy and intuitive. With the "Add Book" feature, you can quickly enter details like the book's title, author, page count, and your reading progress. You can then choose to save the entry, creating a visually appealing card for each book, or cancel if you need to. These book cards are not just informative but also interactive, letting you update your reading progress or remove books from your collection with ease. This project shows how simplicity and efficiency can improve digital library management.

    Tools used: HTML, CSS, and JavaScript

  • 🧮 Calculator ➕

    The Calculator Web Application is a simple and stylish tool that makes doing math easier. It has a clean and easy-to-use interface, with all the basics you need - number buttons, basic math operations, a clear button, and an equal sign to get your answer. You can easily type in numbers and operations, then press the equal sign to see the result right on the screen. This project shows how you can combine functionality and design to create a useful and user-friendly tool for everyday math tasks.

    Tools used: HTML, CSS, and JavaScript
